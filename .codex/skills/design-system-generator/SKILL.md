# Design System Generator

Force design choices to be explicitly chosen and approved.

## Token categories
- typography
- spacing
- colors
- radius
- shadows
- motion
- density

## Workflow
1. Present 2-4 options per token category.
2. Recommend one option.
3. Record the user's selection.
4. Generate a look-lock summary.
5. Treat approved tokens as locked constraints.

## Output
- token reference
- look-lock summary
- developer token block
