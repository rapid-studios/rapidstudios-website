# Mobbin UX Research

Research UI patterns and prepare findings for a visual decision workflow.

## Workflow
1. Clarify component, platform, and goals.
2. Search Mobbin and capture screenshots.
3. Cluster examples by decision area:
   - hero
   - navigation
   - pricing
   - social proof
   - showcase
   - typography tone
   - color tone
   - motion style
4. For each area, prepare:
   - 3-4 strong examples
   - one recommended direction
   - short reasoning
5. Hand results to the decision-board step.

## Output
- markdown research summary
- screenshot set
- per-section recommendations
