# Design Drift Guard

Prevent future inconsistencies after the system is established.

## Checks
- token compliance
- typography scale
- spacing consistency
- color usage
- component consistency
- motion pattern consistency

## Output
- flagged issues
- why they violate the system
- recommended correction
