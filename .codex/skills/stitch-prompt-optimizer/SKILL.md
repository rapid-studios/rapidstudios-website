# Stitch Prompt Optimizer

Optimize prompts before Stitch generation.

## Inputs
- approved UX spec
- approved design direction
- locked token choices
- motion direction
- platform

## Output
- final generation prompt
- visual constraints
- avoid list
- direction summary

## Goal
Reduce drift between approved decisions and generated concepts.
