# Design Decision Board

Create a visual board for collaborative design decisions.

## Board structure
Each section must include:
- decision title
- 3-4 visual examples
- recommended option
- short reasoning
- user choice area

## Example sections
- Hero style
- Navigation style
- Pricing style
- Social proof style
- Showcase style
- Typography direction
- Color direction
- Motion direction

## Rule
Do not move forward until the user has answered the board.
