# Figma Auto-Board Builder

Automatically build Figma boards from structured inputs.

## Supported board types
- Research Board
- Decision Board
- Look-Lock Board
- Concept Board
- Style Guide Board

## Inputs
- project name
- board type
- section list
- screenshots or visual assets
- recommendation notes
- decision notes

## Output rules
For each board:
- create clean section headers
- place screenshots in consistent grids
- add recommendation and rationale
- reserve visible space for user decisions
- keep naming consistent for future reuse
