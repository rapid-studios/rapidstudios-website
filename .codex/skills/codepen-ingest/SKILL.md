# CodePen Ingest

Use this skill to turn CodePen references into implementation support.

## Best use cases
- hero animation references
- hover behaviors
- pricing toggles
- logo clouds
- loaders
- showcase interactions
- nav transitions

## Output
- what the pen does
- structural notes
- motion notes
- implementation caveats
- whether it should be adapted or only referenced
