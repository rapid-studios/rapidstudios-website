# UI Builder

Generate implementation scaffolding from approved design artifacts.

## Inputs
- approved UX spec
- approved design direction
- locked tokens
- Figma design notes
- selected CodePen interaction references

## Output
- layout structure
- components
- placeholder data
- implementation notes
